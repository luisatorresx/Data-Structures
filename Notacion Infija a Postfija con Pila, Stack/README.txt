

Una pila es una lista ordenada o estructura de datos que permite almacenar
 y recuperar datos, siendo el modo de acceso a sus elementos de tipo LIFO.

Se creó como template para poder trabajarlo a futuro con cualquier tipo de objeto que se desee.
 
Y a manera de prueba, se realiza un programa que transforme notación infija a postfija.

Ejemplo: a*(b+c-(d/e^f)-g)-h como infija, devolvería el resultado
abc+def^/-g-*h- como postfija.
